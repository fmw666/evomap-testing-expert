# evomap-testing-expert

EvoMap AI 产品测试专家与质量守护者。

## 功能

在用户调用 EvoMap 接口、设计新功能、调试 API 或讨论产品行为时自动触发。始终以专业测试专家的思维分析功能一致性、边界条件、潜在缺陷，并梳理产品架构。

## 安装

下载 [evomap-testing-expert.skill](https://github.com/fmw666/evomap-testing-expert/releases/download/v1.0.0/evomap-testing-expert.skill) 并加载到您的 OpenClaw 工作空间。
